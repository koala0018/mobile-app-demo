# Mobile App Demo

A minimal Android demo app that shows a counter button on the phone.

## What it does

- Launches a simple screen
- Shows a counter
- Increments the count every time you tap the button

## Build

The repository includes a GitHub Actions workflow that builds a debug APK and uploads it as an artifact.

## Install on a phone

1. Download the `app-debug.apk` artifact from the GitHub Actions run.
2. Copy it to your phone.
3. Allow installs from unknown sources if prompted.
4. Install and open the app.
